Doowit - Task Management App

As a developer, Doowit is the first full stack project I have developed and deployed. But before I started coding, I constructed wireframes, high-fidelity mockups and prototype on Adobe XD to silmulate user flows and UI states. This project was made under time constraints which is why I refrained from adding too many features.

CSS Framework & Library
Frameworks and Libraries make the job of the frontend developer easier. In this particular project, I also happen to be the developer. So, even during the design phase, I was already looking up CSS libraries to serve as a guide in designing UI components for the app. While there are a lot of popular libraries readily available like Bootstrap and Material UI, I went with Flowbite for this project since I was also planning to use Tailwind CSS.

MERN Stack Project
The deployed version of this app is built using the MERN stack— React JS for the frontend, Node.js and Express for the backend, with MongoDB handling the database. It's important to note that the features are still limited and loading screens are not yet added on the deployed site.
